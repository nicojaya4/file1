VPN REMOTE

VPN REMOTE merupakan layanan VPN khusus untuk remote MikroTik dari luar jaringan walaupun anda tidak mempunyai IP Public. Sangat cocok bagi anda yang sering berada di luar jaringan namun ingin tetap dapat memantau jaringan anda. Selain router MikroTik,VPN REMOTE juga dapat digunakan untuk memantau perangkat di bawahnya seperti router ataupun access point, serta berbagai fungsi spesial lainnya.

Fungsi VPN Remote

  1. REMOTE MIKROTIK - VPN REMOTE utamanya ditujukan untuk remote mikrotik dari luar jaringan. Bisa menggunakan winbox, webfig, dan lain-lain.
  2. TUNNEL API MIKROTIK - VPN REMOTE dapat digunakan sebagai tunnel aplikasi yang menggunakan API MikroTik seperti Mikhmon, Mikbotam, dan lain-lain.
  3. REMOTE PERANGKAT LAIN - VPN REMOTE dapat digunakan untuk remote perangkat di belakang Mikrotik, seperti access point, router, web server, dan lain-lain.

Lebih Aman

VPN REMOTE menggunakan protokol yang terenkripsi sehingga koneksi saat anda remote akan lebih aman.

Harga Murah

VPN REMOTE jauh lebih hemat dibandingkan anda harus menyewa IP Public untuk router anda hanya untuk remote.

Server Indonesia

Server VPN REMOTE berlokasi di Indonesia sehingga koneksi akan jauh lebih cepat dibandingkan dengan server di luar Indonesia.
